# CapoMod-Mobile-Tab

![License](https://img.shields.io/badge/license-MIT-blue.svg) [![Latest Stable Version](https://img.shields.io/packagist/v/capomod/mobile-tab.svg)](https://packagist.org/packages/capomod/mobile-tab) [![Total Downloads](https://img.shields.io/packagist/dt/capomod/mobile-tab.svg)](https://packagist.org/packages/capomod/mobile-tab)

A [Flarum](http://flarum.org) extension. CapoMod Custom Mobile Tab

## Installation

Install with composer:

```sh
composer require capomod/mobile-tab:"*"
```

## Updating

```sh
composer update capomod/mobile-tab:"*"
php flarum migrate
php flarum cache:clear
```
